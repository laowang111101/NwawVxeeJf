# Python学生成绩管理系统 python420

本项目是一个基于Python语言开发的学生成绩管理系统，采用Flask框架进行Web开发，使用MySQL数据库进行数据存储，通过HTML进行前端页面展示。

## 技术栈

- 后端：Python
- Web框架：Flask
- 数据库：MySQL
- 前端：HTML

## 功能模块

系统主要包括以下五个模块：

1. 学生信息录入
2. 选课信息录入
3. 成绩信息录入
4. 学生成绩查询
5. 管理员维护

## 系统角色

- 学生：可以进行成绩查询。
- 管理员：负责学生信息、课程信息、成绩信息的录入与维护。

## 运行环境

- 操作系统：兼容多种操作系统
- Python版本：3.x
- 依赖库：Flask、MySQL-Python

## 部署步骤

1. 安装Python环境。
2. 安装Flask框架。
3. 安装MySQL数据库。
4. 克隆或下载项目源码。
5. 根据需求配置数据库连接信息。
6. 运行项目。
